# SyncSpace Socket Events

## Connection
- `connection`: Fired when a client connects.
- `disconnect`: Fired when a client disconnects.

## Room Events
- `join-room`: Sent by client to join a specific room room.

## Whiteboard Events
- `draw-line`: Sent/Received for real-time drawing updates.

## Editor Events
- `editor-update`: Sent/Received for real-time code editor updates.
